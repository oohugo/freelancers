# Projeto Final do Treinadev - Etapa 1

### Como Rodar a Aplicação 

No terminal, clone o projeto. 

	git@github.com:oohugo/freelancers.git
   
Execute o setup

	bin/setup

Adicionar seeds

	rails db:seeds
Rodar os testes

    bundle exec rspec 

### Gems
- rspec
- capybara
- simplecov
- devise
- factory_bot_rails
- rubocop_rails

### Versions
- Ruby version: 3.0.2
- Rails version: 7.0.0
